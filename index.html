AI without Internet

BikeCoach AI works fully offline. Your training plan is generated on-device using built-in rules (safe progression, workout types, rest-day logic), so it does not send your data to any server and does not require an API key.

How we built it

We replaced cloud AI with a local planner: the app collects your goal (speed or distance), your current level, your target, and the training dates you select. Then it generates workouts only for those dates, applies gradual progression, and marks extra sessions as optional once the goal is reached. Everything is stored locally on your phone.

FAQ

Is this the final release?

No. The Android version is currently in beta testing, so UI and features may change. iOS is coming soon.

How do I install the APK?

Download the APK, open it on your phone, and allow installs from unknown sources for your browser or file manager when prompted.

Do I need any API keys for maps?

No. The app uses OpenStreetMap and does not require Google Maps API keys.
